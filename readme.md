# Goals App

A Goals App made using MERN Stack.

## Features

- Multiple user support
- Register New user
- Create Goals (for each user)
- Delete Goals (for each user)

### Install dependencies

```
# Backend deps
npm install
# Frontend deps
cd frontend
npm install
```

### Usage

To be executed from root folder

```
# Run server
npm run server
# Run Frontend
npm run client

# Run both frontend and backend
npm run dev
```
